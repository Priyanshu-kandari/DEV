# loomé. 🌿
A minimal luxury fashion e-commerce product page built with pure HTML & CSS.
 
## Features
- Product page with image gallery & color swatches
- Absolute positioned "Add to bag" circle button
- Active nav underline, trust bar, vertical rotated text
## Typography
- **Cormorant Garamond** — headings & logo
- **Jost** — body & UI text
## Run
Open `index.html` in your browser — no setup needed.
